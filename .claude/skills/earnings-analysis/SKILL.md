# A 股财报分析 (Earnings Analysis)

5 阶段系统化财报分析流程，适配 A 股会计准则和信息披露特点。

## 5 阶段流程

### Phase 1: 数据收集
**数据源**：
| 数据类型 | 主数据源 | 备选 |
|---------|---------|------|
| 三表 (BS/IS/CF) | 东财 datacenter-web | AKShare |
| 年报 PDF | 巨潮资讯网 (cninfo) | 上交所/深交所 |
| 分析师一致预期 | 同花顺 iFinD | 东财盈利预测 |
| 行业对比 | 申万行业分类 | 东财板块 |

**时效性检查**：
- 确认财报为最新一期 (3 个月内)
- 同时获取上个季度和上年同期数据
- 获取 pre-earnings 一致预期 (用于 Beat/Miss 分析)

### Phase 2: 分析
**收入与利润**：
- 营收增速 (YoY / QoQ)
- 扣非净利润增速
- 非经常性损益占比 (A 股重点: 资产处置/政府补贴)
- 分产品/分地区收入拆解

**毛利率与费用**：
- 毛利率趋势 (N 期)
- 费用率 (销售/管理/研发/财务)
- 研发费用资本化比例

**ROE 杜邦分解**：
- 销售净利率 × 资产周转率 × 权益乘数
- 识别 ROE 变动的核心驱动因子

**现金流质量**：
- 经营现金流/净利润 (A 股重点: 应收账款 vs 票据)
- 自由现金流 = 经营现金流 - CAPEX
- 投资现金流 vs 筹资现金流结构

**资产负债表**：
- 商誉/净资产 (A 股重点: > 0.3 警示)
- 应收账款周转天数
- 存货周转天数
- 短期/长期偿债能力

### Phase 3: 图表
- N 期营收/EPS/毛利率趋势
- ROE 杜邦分解图
- 经营现金流 vs 净利润对比
- 资产负债表结构饼图
- 估值带 (PE/PB 历史区间)

### Phase 4: 报告
结构化输出：
1. **执行摘要**: 关键发现 + 评级 + 信心度
2. **收入与利润分析**: 增速 + 驱动力 + Beat/Miss
3. **质量评估**: ROE/现金流/商誉/应收
4. **红旗清单**: 商誉过高/应收异常/关联交易/质押风险
5. **催化剂**: 下季度展望 + 潜在驱动
6. **估值**: PE/PB 分位 + DCF 参考

### Phase 5: 质量控制
- 三表勾稽校验 (A=B?)
- 3 个独立数据源交叉验证 (mootdx/东财/AKShare)
- 置信度校准
- [UNSOURCED] 标记检查

## A 股特定处理

| 项目 | 处理方式 |
|------|---------|
| 非经常性损益 | 扣非净利润优先于归母净利润 |
| 商誉减值 | 商誉/净资产 > 0.3 → 标记风险 |
| 关联交易 | 占比 > 营收 30% → 标记风险 |
| 应收账款 vs 票据 | 区分银行承兑汇票 (低风险) 和商业承兑汇票 (高风险) |
| 研发费用资本化 | 资本化率异常 → 标记 (利润操纵嫌疑) |
| 政府补贴 | 扣非时剔除，评估可持续性 |

## 护栏

- **三个独立数据源交叉验证**
- **所有数字标注来源和日期**
- **财报数据有效期 24h, 超期标记 STALE**
- **不做盈利预测 — 仅分析历史数据和一致预期**
