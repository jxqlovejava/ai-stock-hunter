# 宏观货币信用监控 (Macro & Monetary-Credit Monitor)

基于中国货币政策框架的系统化宏观监控。核心：货币-信用双象限 → 板块影响 → 仓位建议。

## 监控指标

| 指标 | 频率 | 来源 | 作用 |
|------|------|------|------|
| M2 增速 | 月度 | 央行 | 货币供给 |
| 社融存量增速 | 月度 | 央行 | 信用需求 |
| M1-M2 剪刀差 | 月度 | 计算 | 资金活化程度 |
| LPR 1Y/5Y | 月度 | 央行 | 贷款利率方向 |
| DR007 | 日度 | 银行间 | 短期流动性 |
| MLF 利率 | 月度 | 央行 | 政策利率锚 |
| 信贷脉冲 | 月度 | 计算 | 信用周期加速/减速 |

## 货币-信用双象限

```
              信用宽 (社融↑)
                  |
    宽货币+宽信用   |   紧货币+宽信用
   (成长/周期利好)  |   (银行/保险利好)
                  |
   ——————————————+—————————————— 货币
                  |
    宽货币+紧信用   |   紧货币+紧信用
   (公用/必选利好)  |   (现金/防御为王)
                  |
              信用紧 (社融↓)
```

## 象限 → 板块映射

| 象限 | 利好板块 | 利空板块 |
|------|---------|---------|
| 宽货币+宽信用 | 成长 (TMT/新能源)、券商 | 债券 |
| 宽货币+紧信用 | 公用事业、必选消费 | 银行 (息差收窄) |
| 紧货币+宽信用 | 银行、保险 | 高杠杆地产 |
| 紧货币+紧信用 | 现金、防御 (公用/医药) | 成长 (杀估值) |

## 输出格式

### 宏观总览
- 当前象限 + 置信度
- 关键指标快照 (M2/社融/DR007/LPR/信贷脉冲)
- 趋势判断 (加速/减速/稳定)

### 细分项
- M2 增速: 当前值 + 历史分位 + 方向
- 社融增速: 当前值 + 政府债/企业债/信贷结构
- M1-M2 剪刀差: 当前值 + 资金活化信号
- DR007: 当前值 + 偏离政策利率幅度
- LPR: 1Y/5Y 当前值 + 变动方向

### 板块影响
- 利好板块列表 + 逻辑
- 利空板块列表 + 逻辑
- 仓位建议 (高/中/低)

## 护栏

- **数据以央行公布为准**（T0 优先，T1 回退）
- **前瞻性判断需标注置信度**
- **政策转向信号需 2 个以上独立指标确认**
- **信息源质量**：所有宏观指标必须标注 T0-T3 分级 + 时效性，详见 `guardrails.md` 时效性校验规则
