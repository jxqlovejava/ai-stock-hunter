# 选股筛选 (Idea Generation)

系统化 A 股选股。内置 5 种方法论预设，支持全市场扫描和深度分析。

## 5 种筛选预设

| 预设 | 核心逻辑 | A 股适配 |
|------|---------|---------|
| `value` (价值) | PE<15, PB<1.5, 股息率>2%, 低负债 | 排除 ST/*ST，市值>20亿，扣非优先 |
| `growth` (成长) | 营收增速>15%, 利润增速>20%, ROIC>15% | 扣非口径，排除一次性收益，商誉检查 |
| `quality` (质量) | 连续5年盈利, ROE>15%, 低杠杆, 高FCF | 关注关联交易，排除应收异常 |
| `short` (风险识别) | 收入减速, 利润率压缩, 应收异常, 高管减持 | 仅识别风险（A股不做空），提示质押风险 |
| `special-situation` (事件驱动) | IPO锁定期/重组/增发/股权激励/回购 | 适配A股事件类型，锁定期跟踪 |

## 工作流

### Step 1: 选择方法论
用户指定预设或描述选股偏好：
```
/选股 --preset value --limit 20
/选股 "找PE<20且ROE>15%且北向增持的股票"
```

### Step 2: 全市场量化筛选
- 加载预设的阈值和权重配置
- 对全市场股票运行快速排除（ST/新股/低流动性）
- 对通过初筛的股票运行多维分析（L1Analyzer）

### Step 3: 主题相关性扫描
- 检查候选股票是否匹配当前活跃主题
- 主题生命周期阶段修正得分：
  - EMERGING (+10%): 主题刚出现，有 alpha
  - SPREADING (±0%): 正常扩散
  - CONSENSUS (-10%): 共识形成，拥挤风险
  - CROWDED (-20%): 过度拥挤，降权

### Step 4: 排名输出
- 按综合得分降序排列
- 每只股票附带：得分明细、信心度、数据新鲜度
- 前 3 名可选深度分析（L1→L2→L3→L4）

### Step 5: 输出

```
🏆 选股结果 [价值投资] — 前 10 名:

 1. 600519 贵州茅台  综合 82.3 | 价值 75 质量 90 动量 65 宏观 55 | 信心度 0.85
 2. 000858 五粮液    综合 78.1 | 价值 80 质量 82 动量 60 宏观 55 | 信心度 0.82
...

筛选条件: PE<15, PB<1.5, 股息率>2%, 市值>20亿
数据来源: mootdx (2026-07-05 14:30)
时效性: ✅ 数据新鲜
```

## 与管道集成

选股结果可以无缝进入全链路分析：
```
选股 → L1 分析 → L2 裁决 → L3 信号 → L4 风控
```

## 护栏

- **每个数字标注来源**：无法溯源则标记 `[UNSOURCED]`
- **筛选非结论**：所有筛选结果需要基本面验证
- **不做空 A 股**：`short` 预设仅用于风险识别
- **主题拥挤警示**：CROWDED 阶段的主题相关股票降权 20%
- **最佳想法常在交叉点**：质量公司 + 暂时性困境 = 价值机会
