# 政策跟踪 (Policy Tracker)

系统化采集和解析中国政策信号，识别对 A 股板块的影响。

## 覆盖来源

| 来源 | 频率 | 影响等级 |
|------|------|---------|
| 中央政治局会议 | 月度/季度 | ⭐⭐⭐⭐⭐ |
| 国务院常务会议 | 周度 | ⭐⭐⭐⭐ |
| 央行货币政策报告 | 季度 | ⭐⭐⭐⭐ |
| 中央经济工作会议 | 年度 | ⭐⭐⭐⭐⭐ |
| 产业政策 (各部委) | 不定期 | ⭐⭐⭐ |

## 工作流

### Step 1: 政策采集
- 国信证券宏观 API (主)
- 华泰证券市场洞察 (备)
- 政府官网爬取 (补充)

### Step 2: NLP 解析
- 关键词提取 (30 个政策术语 → 板块映射)
- 情绪分析 (利好/利空/中性)
- 影响强度评分 (0-100)

### Step 3: 板块映射

| 关键词 | 映射板块 |
|--------|---------|
| 新能源/碳中和 | 电力设备、新能源 |
| 数字经济/数据要素 | 计算机、通信 |
| 房地产/住房不炒 | 房地产、建材 |
| 注册制/资本市场改革 | 券商、金融 IT |
| 芯片/集成电路 | 半导体 |
| ... 30+ 映射规则 | ... |

### Step 4: 强度评分

```
policy_strength = 
  来源权威度 (0.3) + 措辞力度 (0.3) + 提及频率 (0.2) + 历史兑现率 (0.2)
```

## 输出

```python
PolicySignal(
    title="政治局会议: 活跃资本市场",
    source="中央政治局",
    keywords=["资本市场", "投资者信心"],
    affected_sectors=["券商", "金融IT"],
    sentiment="利好",
    strength_score=85,
)
```

## 护栏

- **以官方文件原文为准**
- **避免过度解读措辞变化**
- **产业政策兑现周期长 (6-12 月)**
