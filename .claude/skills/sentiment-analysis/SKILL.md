# 情绪分析 (Sentiment Analysis)

检测 A 股市场情绪状态，识别极端情绪下的交易机会和风险。

## 信息源质量前置检查（强制 · 不可跳过）

**任何情绪分析必须先执行以下步骤，禁止在未分级的信息上直接输出结论。**

### Step 0.1: 信息条目化

将每条信息拆分为独立条目，标注：
- 来源 provider（eastmoney / mootdx / guosen / akshare / sina / jiemian / nbd / cfi / web_search / llm_derived）
- 发生时间（距现在多少小时）
- 性质 nature：`fact` / `interpretation` / `speculation`

### Step 0.2: T0-T3 信源分级

按 `src/data/source_citation.py` 标准对每条信息分级：

| Tier | 定义 | 示例 | 质量调整 |
|:--:|------|------|:--:|
| **T0** | 一手官方 | 交易所/央行/巨潮公告 | confidence × 1.15 |
| **T1** | 权威数据商 | mootdx/guosen/eastmoney datacenter | confidence × 1.05 |
| **T2** | 聚合/媒体 | akshare/tencent/财经媒体 | 不做调整 |
| **T3** | 推测/三手 | LLM推导/自媒体/传闻/未验证 | confidence × 0.70 |

### Step 0.3: 时效性校验

| 数据类型 | 有效期 | 超期处理 |
|------|:--:|------|
| 实时行情 | 5 min | 过期 → confidence × 0.7 |
| 日线数据 | 1 h | 过期 → confidence × 0.7 |
| 宏观指标 | 4 h | 过期 → confidence × 0.7 |
| 新闻事件 | 6 h | 过期 → 标记 `[STALE]`，**不得作为当日主要归因** |
| 政策/主题 | 12 h | 过期 → confidence × 0.7 |
| 基本面 | 24 h | 过期 → confidence × 0.7 |

> ⚠️ **关键规则**：发生时间超过 freshness limit **2 倍以上**的新闻事件，必须标记 `[STALE]`，**排除出当日归因列表**，仅可作为背景提及。例如：7天前的新闻不能解释今天的下跌。

### Step 0.4: 交叉验证

- 关键盘面数据（涨跌家数/成交额/北向资金）必须 ≥ 2 个独立来源
- 单源数据 → 标注 `⚠️ 单源未验证`，对应结论 confidence 降 0.1
- 两源差异 > 3% → 标记 `[DISPUTED]`，不进入归因

### Step 0.5: 输出信息源质量报告

分析输出前必须包含：

```
📋 信息源质量总览
| Tier | 数量 | 平均质量分 |
| T0   | N    | X.XXX      |
| T1   | N    | X.XXX      |
| T2   | N    | X.XXX      |
| T3   | N    | X.XXX      |
交叉验证率: XX%
整体 confidence: X.XX
过期信息(已排除): [列出 STALE 条目]
[DATA_GAP]: [列出数据缺口]
```

### 信息可得性分级（ai-berkshire news-pulse 框架）

| 等级 | 特征 | 策略 |
|:--:|------|------|
| **A 级** | 大盘、媒体覆盖广、数据丰富 | 重点是降噪——区分 T0-T1 与 T2 转述 |
| **B 级** | 中小盘、覆盖一般 | 每条关键事件需 1-2 个独立信源 |
| **C 级** | 冷门标的、信息稀缺 | "找不到解释"本身有价值——可能是技术面驱动 |

## 情绪检测

| 情绪 | 信号 | 市场含义 |
|------|------|---------|
| EXTREME (极度恐慌) | VIX 飙升/跌停潮/ETF 折价 | 反向指标 → 可能见底 |
| PANIC (恐慌) | 大幅下跌/放量/止损潮 | 不追跌，等待企稳 |
| NORMAL (正常) | 正常波动 | 正常交易 |
| GREED (贪婪) | 连涨/缩量上涨/涨停潮 | 谨慎追高 |

## 恐慌套利 (Panic Arbitrage)

当市场出现非理性恐慌时：
1. 检测恐慌信号 (跌幅/放量/情绪指标)
2. 寻找被错杀的优质标的 (PE 低估 + ROE 稳定 + 现金流健康)
3. 分批建仓 (不抄底，等企稳信号)
4. 设置紧密止损 (-1.5% vs 正常 -2%)

## 成交量异常检测

- 放量上涨 → 突破确认
- 放量下跌 → 恐慌出逃
- 缩量上涨 → 上涨乏力
- 缩量下跌 → 杀跌动能衰竭

## 涨跌停比例

| 比例 | 含义 |
|------|------|
| 涨停/跌停 > 3:1 | 极度乐观 |
| 3:1 > 涨停/跌停 > 1:1 | 正常 |
| 涨停/跌停 < 1:3 | 极度悲观 |

## 护栏

- **恐慌中不抄底**: 等待成交量萎缩 + 跌幅收窄
- **贪婪中不追高**: 检查估值和换手率
- **情绪指标是辅助**: 不与基本面分析矛盾
