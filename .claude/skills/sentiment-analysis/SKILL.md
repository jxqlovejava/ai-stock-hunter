# 情绪分析 (Sentiment Analysis)

检测 A 股市场情绪状态，识别极端情绪下的交易机会和风险。

## 情绪检测

| 情绪 | 信号 | 市场含义 |
|------|------|---------|
| EXTREME (极度恐慌) | VIX 飙升/跌停潮/ETF 折价 | 反向指标 → 可能见底 |
| PANIC (恐慌) | 大幅下跌/放量/止损潮 | 不追跌，等待企稳 |
| NORMAL (正常) | 正常波动 | 正常交易 |
| GREED (贪婪) | 连涨/缩量上涨/涨停潮 | 谨慎追高 |

## 恐慌套利 (Panic Arbitrage)

当市场出现非理性恐慌时：
1. 检测恐慌信号 (跌幅/放量/情绪指标)
2. 寻找被错杀的优质标的 (PE 低估 + ROE 稳定 + 现金流健康)
3. 分批建仓 (不抄底，等企稳信号)
4. 设置紧密止损 (-1.5% vs 正常 -2%)

## 成交量异常检测

- 放量上涨 → 突破确认
- 放量下跌 → 恐慌出逃
- 缩量上涨 → 上涨乏力
- 缩量下跌 → 杀跌动能衰竭

## 涨跌停比例

| 比例 | 含义 |
|------|------|
| 涨停/跌停 > 3:1 | 极度乐观 |
| 3:1 > 涨停/跌停 > 1:1 | 正常 |
| 涨停/跌停 < 1:3 | 极度悲观 |

## 护栏

- **恐慌中不抄底**: 等待成交量萎缩 + 跌幅收窄
- **贪婪中不追高**: 检查估值和换手率
- **情绪指标是辅助**: 不与基本面分析矛盾
