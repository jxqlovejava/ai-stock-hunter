# DCF 估值工作流

## 概述

DCF（自由现金流折现）是一套严密的绝对估值方法。本工作流由 7 步构成，覆盖完整估值链路。

## 步骤一：收集历史自由现金流

1. 获取最近 5-7 年财报数据
2. 计算历年自由现金流 FCF = 经营活动现金流净额 - 资本开支
3. 排除一次性/非经常性项目干扰
4. 输出：逐年 FCF 时序表

### 关键检查

- [ ] 经营现金流净额是否含非经常性收入
- [ ] 资本开支是否与折旧匹配
- [ ] 自由现金流是否连续覆盖融资成本
- [ ] 若某年 FCF 为负，确认原因（扩张期 vs 经营恶化）

## 步骤二：计算历史增长率

1. 计算 CAGR（3 年 / 5 年）
2. 计算各年同比增速
3. 计算增速中位数与均值
4. 分析增速稳定性与趋势

### 数据质量

- 历史增速波动过大 → 降低 confidence
- 某年增速异常（>100% 或 <-80%）→ 确认是否一次性事件
- 必要时调整剔除 outlier 年份

## 步骤三：估算 WACC（加权平均资本成本）

WACC = E/V * Ke + D/V * Kd * (1 - T)

### 权益成本 Ke

Ke = Rf + beta * ERP

- **Rf（无风险利率）**：取 10 年期国债收益率（当前约 2.2%-2.8%）
- **Beta**：取 2-5 年历史 beta（参考申万行业 beta）
- **ERP（股权风险溢价）**：A 股建议 6%-7.5%（历史均值约 6.5%）

### 债务成本 Kd

- 取公司最近一期有息负债平均利率
- 如无法获取，用同评级企业债收益率 + 200-400bp

### 资本结构

- 按市值计算权益权重
- 按账面有息负债计算债务权重

### WACC 敏感性

- 基准 WACC ± 0.5% / ± 1.0% 各算一档
- 输出 5 档 WACC 对应的估值范围

## 步骤四：5 年自由现金流预测

1. 以最近一年 FCF 为基数
2. 第 1-2 年：取最近增速中位数
3. 第 3-5 年：线性衰减至永续增长率 g
4. 每年计算折现因子 1/(1+WACC)^n
5. 输出逐年折现 FCF 及累计值

### 预测衰减模型

```
Year 1-2:  g1 = median(growth_rate)
Year 3:    g3 = (g1 + g_terminal) / 2
Year 4:    g4 = (g1 + 3*g_terminal) / 4
Year 5:    g5 ~ g_terminal
```

## 步骤五：终值计算

### Gordon 永续增长模型

终值 = FCF_5 * (1 + g) / (WACC - g)

### g（永续增长率）取值规则

- 保守：g = 长期通胀率 ≈ 2%-3%
- 中性：g = GDP 增速预期 ≈ 3%-4%
- 激进：g = 行业长期增速（不超过 WACC - 2%）

### 终值占比检查

终值占内在价值比重 > 70% → 结果高度依赖假设，需下调整体 confidence

- 最坏：WACC +1%、g 2%、FCF 衰减加速
- 基准：WACC 基准、g 3%、正常预测
- 最好：WACC -1%、g 4%、FCF 增长超预期

## 步骤七：内在价值 vs 市场价格

1. 总企业价值 = 折现 FCF 之和 + 终值折现值
2. 股权价值 = 总企业价值 - 净债务 + 非核心资产
3. 每股内在价值 = 股权价值 / 总股本
4. 安全边际 = (内在价值 - 当前股价) / 当前股价

### 投资判断

| 安全边际 | 含义 | 行动 |
|-----------|------|------|
| > 30%     | 显著低估 | 可考虑建仓 |
| 15%-30%   | 适度低估 | 继续跟踪 |
| -15%-15%  | 合理估值 | 持有不动 |
| < -15%    | 高估     | 等待回调 |
