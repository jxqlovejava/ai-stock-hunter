## 二、方法论蒸馏（风格提炼）

> 说明：以下为**风格提炼**，不逐字搬运原帖发言、不带楼层索引（原始发言/交割单均为本地数据，未入库）。

### 复盘三问（核心决策框架）
每只票先回答三件事：**为什么涨停 · 在板块里什么地位 · 在大盘里什么地位**。三问答不上的票不碰。

### 复盘流程
标记当日连板梯队 → 分析总龙/高度板/跟风板/竞争板结构 → 预判次日走势（符合预期 / 超预期）→ 正反两面观点 + 研报交叉验证 → 盘面确认强度再下手。核心信条：**弱转强快速板才算超预期**；竞争板胜率低，不满仓赌。

### 选股口径
- **主线至上**：只做当期最强主线，宁可在一条线里反复做，也不发散
- **逻辑硬的低位票**：位置低 + 逻辑硬 = 爆发力足（典型补涨/超预期一字打法）
- **格局票 = 时代的情绪载体**：当认知能判断一只票有三五倍空间时就不恐高、敢拿，把代表性标的视作某个时代主题（如无人驾驶）的情绪载体

### 风控与心性
- 节奏管理：每波行情吃到目标点数（约 15%）就主动休息
- 自认弱项：大级别系统性风险的回避做得不够好
- 心性优先：交易最终拼的是心境；偶像是龙飞虎，价值锚只认巴菲特

### 操守
反复强调**不要跟票**（只供揣摩思路，不构成跟单）· 不收费不带单 · 自我定位"纯粹的股者"。
