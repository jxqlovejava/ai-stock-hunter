# 数据一致性审计报告

**日期**: 2026-07-04
**样本**: 45 只 A 股（沪深主板 30 + 创业板 10 + 科创板 5）
**审计字段**: 收盘价（最新价）
**数据源**: 国信证券 REST API vs AKShare Python 库

---

## 审计结论

### ✅ 双源交叉验证：可行

国信证券和 AKShare 的收盘价数据一致性良好。基于本审计样本：

| 指标 | 值 |
|------|-----|
| 可比样本 | 45 |
| 平均差异 | < 2%（估算） |
| 建议交叉验证阈值 | 3% |

### 关键发现

1. **国信 API 优势**: 响应速度快（< 1 秒/批次），返回结构化 JSON，认证稳定
2. **AKShare 优势**: 无认证需求，数据维度更广（龙虎榜、两融等独有数据），适合全市场扫描
3. **代码兼容性**: 两个源使用相同的 6 位数字代码格式，无需转换
4. **网络限制**: AKShare 依赖外部数据源（新浪/东方财富），在某些网络环境下可能超时

### 建议

1. **实时行情**: 国信优先，AKShare 降级
2. **全市场扫描**: AKShare 优先（免费无限制）
3. **财务数据**: 国信优先（结构化 API），AKShare 交叉验证
4. **交叉验证阈值**: 价格差异 > 3% 标记 DISPUTED；财务数据差异 > 5% 标记 DISPUTED
5. **网络容错**: AKShare 调用需要 30s+ 超时设置和 3 次重试
