> **实验目的**：假设时间回到 2025年6月1日，使用白泽系统现有选股流水线，筛选 AI 科技赛道的高价值与低价值标的。
> **后续验证**：对比 2025.6.1 → 2026.7.9 实际表现，评估选股能力。

## 一、选股流水线速查（14 步）

```
数据获取 → 军规(34条) → 准入(ST/次新/流动性) → 增强上下文(宏观/北向/主题)
→ 基本面三问 → Alpha Lens(4维) → 反操纵扫描 → 多维诊断(8+维度)
→ 博弈论分析 → 投资思维模型 → 四大师辩论 → T+0日内 → 综合裁决 → 仓位风控
```

### 核心评分公式

```
fundamental = (value_score + quality_score) / 2
raw_score = fundamental×0.30 + valuation×0.15 + momentum×0.15
          + macro×0.10 + cycle×0.10 + sector×0.10
          + sentiment×0.05 + executive×0.05

final = raw × alpha_multiplier × game_theory_multiplier
      × manipulation_discount × topic_lifecycle_adjustment
```

### 推荐映射

| 分数 | 建议 |
|------|------|
| ≥75 | BUY |
| ≥60 | ADD |
| ≥40 | HOLD |
| ≥25 | REDUCE |
| <25 | SELL |

### 关键约束
- **5日涨幅 > 20%** → 分数封顶 55，强制 HOLD（追涨熔断）
- **confidence < 0.55** → 阻止进入仓位调度
- **主题 CROWDED** → sector 得分 ×0.8
- **操纵风险 > 60** → 总分 ×0.7
- **PE 周期感知**：复苏/低谷期高 PE 不线性惩罚，可获 cycle boost
