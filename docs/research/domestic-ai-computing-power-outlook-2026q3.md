# 国产AI算力扩容是否中长期大方向？——综合研究报告

*生成日期：2026-07-11 | 调研来源：30+ 公开来源 | 置信度：中-高（核心事实多源交叉验证，部分预测为单源估算）*

---

## 核心结论

**国产AI算力扩容是一个真实、政策驱动、跨3-5年以上的中长期大方向，但不是“普涨式”牛市。** 最可能的图景是**结构性分化**：

- ** sovereign 基础设施层**（国家枢纽、运营商、华为昇腾生态、政府/央企采购）具备最高确定性和政策托底；
- **商业数据中心层**（地方政府重复建设、 speculative AIDC、高估值国产芯片股）面临产能过剩、估值泡沫和利用率不足的风险；
- **配套产业链**（液冷、光模块、电源、PCB）随AI服务器功耗与带宽升级同步受益，但需甄别客户结构。

**判断为中长期大方向的核心依据：**
1. 政策写入“十五五”规划，定位为科技自立自强核心支撑；
2. 美国对华高端AI芯片出口管制构成“强制国产替代”，国产芯片市占率从2021年约5%升至2025年约40%；
3. 互联网大厂资本开支进入军备竞赛，BAT+字节2025-2026年AI基础设施投入达数千亿人民币；
4. 大模型训练→推理的需求演进，推理算力需求在2025年快速逼近训练算力。

**主要风险：** 先进制程/EDA/HBM/先进封装仍被卡脖子；CANN生态与CUDA差距仍在；部分地方政府智算中心利用率低（约20-30%）；国产GPU股市盈率/市销率极高，已透支多年增长。

## 1. 市场供需：结构性紧平衡，而非简单短缺

### 1.1 需求端四重驱动

| 驱动因素 | 关键事实 | 来源级别 |
|---------|---------|---------|
| **大模型训练与推理** | 2023-2028年中国智能算力CAGR预计46.2%，远超通用算力18.8%；2025年推理算力支出首次逼近训练算力 | IDC/浪潮（二手权威） |
| **企业AI应用落地** | 截至2025年初，35家央企已部署66个大模型应用；能源、金融、政务垂直场景快速接入DeepSeek | 财新/36氪（二手） |
| **互联网/云厂商资本开支** | 阿里未来3年投入超3,800亿元；字节2025年资本开支约1,500-1,600亿元；腾讯2024年资本开支同比+221% | 各公司公告/雪球/媒体转述（二手-三手） |
| **政府智算中心建设** | 2025年地方政府智算中心建设规模预计107.1 EFLOPS，同比+300%；2024年上半年智算中心招标791起，同比+407% | 天风证券/易秀（二手） |

**因果逻辑：** 大模型能力进化（DeepSeek等）→ 降低应用门槛 → 企业/政府私有化部署需求爆发 → 推理算力需求上升 → 国产芯片适配完成 → 国产算力采购加速。这是一个**自我强化的正反馈**，但前提是模型持续迭代和应用真实付费。

### 1.3 供需关系：结构性错配

**不是全面短缺，而是“高端紧缺、低端过剩”：**

| 维度 | 状态 | 关键数据 |
|------|------|---------|
| 全国智算中心平均利用率 | 偏低 | 约20-30%，部分仅10-15% |
| 高端万卡训练集群 | 紧缺 | 互联网大厂仍面临合格集群供给不足 |
| 分散式地方智算中心 | 过剩风险 | 很多1000P项目找不到租户，GPU利用率<30% |
| 区域错配 | 显著 | 东部有需求但缺电，西部有电但需求弱，仅约30%中心支持跨省调度 |
| 算力租赁价格 | 下行 | 从2024年约12元/卡小时降至2025年约9元/卡小时 |

**交叉验证：** 多源（CAICT、IDC、SegmentFault、中研普华）均指出利用率偏低与高端紧缺并存，该判断置信度较高。

## 4. 政策与地缘：国家战略+出口管制双重推力

### 4.1 政策驱动

- **“十五五”规划（2026-2030）**将“科技自立自强”列为核心目标，首次明确要求“加强算力设施支撑”；
- **“AI+”行动计划**从“持续推进”升级为“深化拓展”，标志从孵化进入规模化商用；
- **“东数西算”**八大枢纽、十大集群，要求2025年基本建成算力基础设施体系；
- 传闻中国将投入约**2万亿元人民币（约2,950亿美元）**用于未来五年数据中心和算力枢纽建设；
- 政府采购对国产产品给予约**20%价格折扣**评估优惠，数据中心部署国产处理器可享受最高50%能源补贴。

### 4.2 地缘政治：美国出口管制

| 事件 | 时间 | 影响 |
|------|------|------|
| 美国要求NVIDIA H20对华出口需许可证 | 2025年4月 | NVIDIA计提约151亿美元相关损失 |
| NVIDIA中国AI芯片市占率从95%降至55-60% | 2021-2025 | 强制国产替代窗口打开 |
| 中国暂停镓/锗/锑出口管制（暂定于2026年11月到期） | 2025年 | 关键矿产博弈 |
| 中美关税休战协议到期 | 2026年11月10日 | 潜在升级风险 |

**因果链：** 美国出口管制 → 高端GPU供给受限 → 国产芯片获得强制市场空间 → 华为昇腾/寒武纪/海光订单爆发 → 服务器/IDC/配套产业链受益。

## 6. 中长期判断：是大方向，但非普涨

### 6.1 是否是3-5年以上大方向？

**是。** 理由：
1. 政策写入十五五规划，具有国家战略级别持续性；
2. 国产替代是被动刚需，不会因短期市场波动而逆转；
3. 大模型应用仍处于早期，推理算力需求远未见顶；
4. 互联网大厂资本开支周期通常持续3-5年。

### 6.2 但不是普涨

**分化将加剧：**
- **赢家：** 华为昇腾生态、寒武纪（有大客户订单）、海光（类CUDA兼容）、头部液冷/光模块/电源供应商、运营商/国家队AIDC；
- **承压方：** 无客户/无订单的地方智算中心、高估值但无业绩的GPU概念股、过度依赖单一客户的小供应商。

### 6.3 关键验证指标

| 类别 | 验证指标 |
|------|---------|
| **政策执行** | 十五五规划落地、国家统一算力网建设、东数西算利用率 |
| **国产芯片份额** | 华为昇腾市占率是否突破50%、政府集采国产化率 |
| **技术差距** | SMIC良率提升、国产DUV/EUV进展、国产HBM量产 |
| **软件生态** | CANN开源 adoption、原生昇腾模型数量、大集群稳定性 |
| **需求/利用率** | 数据中心利用率、云AI收入增速、大模型API收入 |
| **能源** | PUE达标、绿电采购、电网接入进度 |
| **财务健康** | 资本开支/收入比、地方数据中心开发商债务、估值合理性 |

### 6.4 什么会证伪该趋势？

- 大规模智算中心项目取消或延期；
- 数据中心利用率持续低于60%或租赁价格持续下跌；
- 美国放松管制导致NVIDIA重新夺回大量份额；
- 国产芯片大集群训练频繁失败；
- 地方政府数据中心开发商出现债务危机。
